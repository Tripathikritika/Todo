# Todo
> A simple input bar to add Todo(s) on pressing the return-key. A list of Todo cards is been appended on creation. Completed task is been segregate as different list of items.
 
## Features
- [x] Adding a new task 
- [x] Reset the todo list
- [x] Completed Task
- [x] Highlighting the card in case of hastag matched
